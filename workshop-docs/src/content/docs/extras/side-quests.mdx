## Real-time with SSE

This is where Datastar really shines, but you can do it with HTMX too.

### HTMX SSE Extension

```html title="SSE with HTMX"
<div hx-ext="sse"
     sse-connect="/events"
     sse-swap="message">
    Waiting for updates...
</div>
```

### Server-side (Flask)

```python title="app.py - SSE endpoint"
from flask import Response
import time

@app.route('/events')
def events():
    def generate():
        while True:
            # Check for new data
            yield f"data: <div>Update at {time.time()}</div>\n\n"
            time.sleep(5)
    return Response(generate(), mimetype='text/event-stream')
```
